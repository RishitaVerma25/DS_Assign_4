- **BST** (Binary Search Tree): insert, search, delete, and inorder traversal

## Project structure

- `BST/bst.py`
- `Graph/graph.py`
- `BFS_DFS/traversals.py`
- `HashTable/hash_table.py`

## How to run

### 1) BST
```bash
python BST/bst.py
```

### 2) Graph (BFS/DFS)
```bash
python Graph/graph.py
```

### 3) Traversals (standalone demo)
```bash
python BFS_DFS/traversals.py
```

### 4) Hash table
```bash
python HashTable/hash_table.py
```

## Notes
- Graph uses an **adjacency list** representation.
- BFS/DFS print traversal order for demonstration.
- Hash table uses **separate chaining** (buckets store lists of key/value pairs).
